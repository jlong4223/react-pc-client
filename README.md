# React.js Port-Chaveriat

This is an application that will be used for scheduling family lake-house events.

## Technologies Used

- React.js
- Node.js, MySQL, Express.js API
- React Router
- Styled Components
- Bootstrap
- Font Awesome
- Google Maps
- Open Weather Map API
- Geolocation
- JsonWebToken
- moment.js
- react-big-calendar
