# React.js Port-Chaveriat

This is an application that replicates a state/national park website.

## Technologies Used

- React.js
- Node.js, MySQL, Express.js API
- React Router
- Styled Components
- Bootstrap
- Font Awesome
- Google Maps
- Open Weather Map API
- Geolocation
- JsonWebToken
